# InvoiceMaker Pro

**InvoiceMaker Pro** is a professional, full-stack invoice and receipt generator designed for small businesses and freelancers. Built with a modern tech stack (React, Node.js, PostgreSQL), it offers a seamless experience for managing clients, products, and invoices.

## Features

- **Dashboard**: Real-time overview of your business performance.
- **Client Management**: Add, edit, and delete client details.
- **Product/Service Management**: Manage your inventory or service offerings.
- **Invoice Generation**: Create professional PDF invoices in seconds.
- **PDF Export**: Download invoices directly to your device.
- **Authentication**: Secure login and registration system.
- **Responsive Design**: Works perfectly on desktop and mobile.

## Tech Stack

- **Frontend**: React (Vite), TailwindCSS, Lucide Icons
- **Backend**: Node.js, Express, Sequelize ORM
- **Database**: PostgreSQL
- **PDF Generation**: PDFKit

## Quick Start

### Prerequisites
- Node.js (v14+)
- PostgreSQL

### Installation

1.  **Clone the repository**
2.  **Setup Backend**
    ```bash
    cd backend
    npm install
    # Configure .env file (see docs/INSTALLATION.md)
    npm start
    ```
3.  **Setup Frontend**
    ```bash
    cd frontend
    npm install
    npm run dev
    ```

For detailed instructions, please refer to the `docs/` folder.
